# Free4Talk
## Run
- Install `yarn`
```
yarn install
```
- Run script
```
node . // or node index.js
```
- Input the room's `token` and get your thing :)))
## How to get the room's token
- Inspect the F4T source code and take a look at the Network tab as the attached image bellow.
